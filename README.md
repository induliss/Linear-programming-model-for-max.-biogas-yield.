# Linear-programming-model-for-max.-biogas-yield.

This repository contains models and methods for optimizing biogas (biomethane) production from lignocellulosic household waste, with a focus on paper-based substrates.

## 🔬 Research Context

This work is part of a doctoral research study focused on:
- maximizing biogas yield
- optimizing substrate composition
- applying mathematical and linear programming methods

## ⚙️ Features

- Linear programming-based optimization model
- Substrate composition analysis
- Experimental data integration (optional)
- Scalable for different reactor sizes

## 📊 Applications

- Biogas plant optimization
- Waste management systems
- Research and academic use
- Industrial feasibility studies
